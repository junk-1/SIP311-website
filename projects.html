StackedFins

Student Innovation Project — In Progress

View on GitHub →

A mobile fishing app that divides Phoenix-area waters into fixed geographic zones, verifies catches through in-app geotagged photos, and awards displaceable, username-only zone ownership to the angler with the most verified catches. A private journal correlates each angler's own log against public USGS, solunar, and weather data to surface personal fishing patterns — without ever sharing a fishing spot or user data between anglers.

React Native Python SQL / SQLite GeoJSON / Geospatial Public Data APIs (USGS, Solunar) Mapbox
